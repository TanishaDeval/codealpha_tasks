# ABOUT
Name : Hangman Game
This is python-based🐍 word guessing game where the player guesses letters to reveal the hidden word,ex- _ _ a _ e_.
# Features
-Random word selection
-Replaces dashes with letter after each correct guess.
-Displays the hangman figure after every wrong guess.
-Input validation with warning messages⚠️.
-Play again option.
-Interactive emoji-based interface.
-Win🏆/lose😞 result display.
# How to play
-Guess one letter from a-z.
-You can make only 6 wrong guesses.
-Correct guesses reveal the hidden word.
-At the end of the game,a win or lose message will be is displayed.
-If you want to play again, enter "y".
-To end the game, enter "n".
