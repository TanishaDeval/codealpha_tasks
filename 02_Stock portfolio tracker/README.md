📈 Stock Portfolio Tracker

## 📖 About the Project

A Python project that helps users calculate their stock investment value by entering stock names and quantities. 💰

## ✨ Features

- 📊 Shows available stocks
- ⌨️ Takes stock name and quantity as input
- ⚠️ Handles invalid stock names
- 🛡️ Handles invalid quantity inputs using try-except
- 🧮 Calculates total investment value
- 📄 Saves the result in a text file

## 🛠️ Concepts Used

- 📚 Dictionaries
- ⚙️ Functions
- 🔄 Loops
- 🚨 Exception Handling
- 📂 File Handling

## ▶️ How to Run

1. 💻 Open the project in PyCharm or any Python IDE.
2. ▶️ Run the Python file.
3. ✍️ Enter stock details.
4. 📈 View the total investment value.
