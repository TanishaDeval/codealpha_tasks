##  Email Extractor

## 📌 About Project

This project extracts email addresses from a TXT file and saves them into another TXT file.

## 🚀 Features

- Read data from a TXT file 📂
- Find email addresses using Regex 🔍
- Remove duplicate emails ✨
- Save extracted emails in a new TXT file 💾


## 🛠️ Technologies Used

- Python🐍
- Regex ("re" module)
- File Handling

## 📁 Files Used

- "Details.txt" → Contains text and email addresses
- "emails.txt" → Stores extracted emails
- "email extractor.py" → Python program

## ▶️ How It Works

1. Read the content of "Details.txt" 📖
2. Find email addresses using Regex 🔍
3. Remove duplicate emails using a set ✨
4. Save unique emails in "emails.txt" 💾

## 📚 What I Learned

- File Handling in Python 📂
- Regular Expressions (Regex) 🔍
- Reading and Writing Files ✍️
- Working with Sets to remove duplicates ✨

## 🎯 Output

The program extracts all email addresses from the Details file and saves them into a separate file without any duplicates.
